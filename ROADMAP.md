# CommonCoin (COM) Development Roadmap

## Phase 1: Core Development & Genesis Mining
* Fork Dogecoin Core v1.14.9
* Apply complete rebranding to CommonCoin (COM)
* Setup custom ports, magic bytes, and address prefixes ('C' for P2PKH)
* Mine Mainnet, Testnet, and Regtest genesis blocks
* Compile Linux/Windows binaries

## Phase 2: Infrastructure & Pool Deployment
* Automate VM provisioning with Terraform on Oracle Cloud Always Free
* Set up Node Open Mining Pool (NOMP) with Redis back-end
* Deploy seed nodes and bootstrap sync nodes
* Deploy Prometheus and Grafana telemetry monitors

## Phase 3: Public Testnet Validation
* Launch public testnet
* Verify peer synchronization, mining, and transaction validation
* Test Qt wallet functionality across platforms

## Phase 4: Mainnet Launch & Community Release
* Deploy mainnet seed nodes and public mining pool
* Launch official landing website and documentation portal
* Publish release binaries and setup guides
