About CommonCoin

Built on Trusted Dogecoin Consensus

CommonCoin is a complete autonomous fork of Dogecoin Core (v1.14.9), maintaining the exact Scrypt hashing algorithm, block rewards, and consensus rules. We've established a completely independent network with a fresh genesis block, decoupled network ports, and custom address schemes to create a pure mining environment.

How To Mine

Start Mining CommonCoin

Since CommonCoin uses the Scrypt proof-of-work algorithm, you can mine it using standard CPU miners, GPU miners, or specialized ASIC devices compatible with Scrypt.

Step 1: Get an Address
Open your CommonCoin Core wallet and create a new receiving address. It will begin with the letter 'C' (e.g., CRx...).
Step 2: Choose a Miner
For CPU mining, use cpuminer-multi. For GPU mining, use cgminer or sgminer with Scrypt support.
Step 3: Connect to Pool & Mine
Configure your mining command pointing to our stratum pool or set up solo mining in your wallet configuration.
miner_commands.sh

# Mine using cpuminer-multi against our stratum pool

minerd -a scrypt -o stratum+tcp://pool.commoncoin.org:3032 -u CYourWalletAddressHere -p x

# Mine using cgminer on opencl graphics cards

cgminer --scrypt -o stratum+tcp://pool.commoncoin.org:3256 -u CYourWalletAddressHere -p x -I 13

# Run local daemon configured for solo mining

commoncoind -daemon -rpcuser=user -rpcpassword=pass -gen=1 -genproclimit=4
