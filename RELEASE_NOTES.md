# Release Notes - CommonCoin v1.0.0-beta

Welcome to the initial pre-release of CommonCoin (COM) - The People's Cryptocurrency.

## Highlights
- **Dogecoin Base**: Built upon Dogecoin Core v1.14.9 consensus.
- **Custom Network Address Prefix**: Addresses start with 'C'.
- **Independent Infrastructure**: Configured for deployment on Oracle Cloud Always Free ARM64 VMs.
- **NOMP Mining Pool integration**: Ready-to-deploy configs for worker mining.
