CommonCoin Core version 1.14.9 is now available from:

  <https://github.com/commoncoin/commoncoin/releases/tag/v1.14.9/>

This is a new minor version release that contains important bugfixes. CommonCoin
users on any platform are strongly recommended to upgrade.

Please report bugs using the issue tracker at GitHub:

  <https://github.com/commoncoin/commoncoin/issues>

To receive notifications about updates, subscribe to the release mailing list:

  <https://sourceforge.net/projects/commoncoin/lists/commoncoin-releases>


Compatibility
==============

CommonCoin Core is extensively tested on Ubuntu Server LTS, macOS and Windows.
Minimum OS compatibility can be found [in the INSTALL guide](../INSTALL.md).

Notable changes
================

Important updates have been added that solve upstream bugs from Bitcoin Core
and Namecoin Core that were inherited by CommonCoin Core.

Known Issues
=============

Compiling on FreeBSD 14.1 may cause an error with endian-specific functions. See
<https://github.com/commoncoin/commoncoin/issues/3673> for a full description and
workaround.
